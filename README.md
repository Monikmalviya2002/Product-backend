# Product Backend

This is the backend server for the **Product Management Application**.  
It provides RESTful APIs for user authentication, OTP verification, and product management.

## Tech Stack
- Node.js
- Express.js
- MongoDB / Mongoose
- JWT Authentication
- Axios (for internal requests)
- Cors, Cookie-Parser

---

## Features
- User registration and login
- OTP verification
- Resend OTP
- CRUD operations for products
- JWT-based authentication with cookies
